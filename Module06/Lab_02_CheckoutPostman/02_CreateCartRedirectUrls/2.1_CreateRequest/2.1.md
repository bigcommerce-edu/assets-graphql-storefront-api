Request name:
GQL Get Cart Redirect URLs

Disable cookie jar:
On

Authorization:
Inherit auth from parent

Request method:
POST

URL:
```
https://store-{{store_hash}}-{{storefront_channel_id}}.mybigcommerce.com/graphql
```

Headers:

| Header       | Value            |
|--------------|------------------|
| Accept       | application/json |
| Content-Type | application/json |


Body:
```
mutation GetCartRedirectUrls(
    $cartId: String!
) {
    cart {
        createCartRedirectUrls(
            input: {
                cartEntityId: $cartId
            }
        ) {
            redirectUrls {
                redirectedCheckoutUrl
                embeddedCheckoutUrl
            }
        }
    }
}
```

Variables:
```
{
    "cartId": "{{cart_id}}"
}
```
