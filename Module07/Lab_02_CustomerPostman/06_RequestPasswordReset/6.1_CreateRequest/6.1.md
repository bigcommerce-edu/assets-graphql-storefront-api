Request name:
GQL Request Password Reset

Disable cookie jar:
On

Authorization:
Inherit auth from parent

Request method:
POST

URL:
```
https://store-{{store_hash}}-{{storefront_channel_id}}.mybigcommerce.com/graphql
```

Headers:

| Header           | Value            |
|------------------|------------------|
| Accept           | application/json |
| Content-Type     | application/json |

Body:
```
mutation ChangePassword(
  $email: String!,
  $redirectPath: String
) {
  customer {
    requestResetPassword(
      input: {
        email: $email,
        path: $redirectPath
      }
    ) {
      errors {
        ... on ValidationError {
          path
          message
        }
      }
    }
  }
}
```

Variables:
```
{
    "email": "{{customer_email}}",
    "redirectPath": "/my-password-reset"
}
```
